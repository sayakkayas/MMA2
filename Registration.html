About

Muscle Memory Authenticator

Muscle Memory Authenticator is aimed at providing authentication using a muscle waves generated while typing.The project tries to provide authentication without users actually trying to remember passwords or other authentication metric.The project has been created by a fun loving group at University of Rochester.
